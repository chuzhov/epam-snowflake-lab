# EPAM Snowflake Hands-on Lab — Project Summary

A hands-on coverage of Snowflake's core and advanced feature set — Snowpipe, Streams, Tasks, Dynamic Tables, External Tables, Stored Procedures, Time Travel, Zero-copy Cloning, and Secure Data Sharing — grounded in a realistic CRM export pipeline backed by Azure ADLS Gen2.

**[View the project summary page →](https://chuzhov.github.io/epam-snowflake-lab/)**

## Stack

Azure ADLS Gen2 · Azure Functions · Azure Event Grid · Snowflake (AWS-hosted) · Power BI · TPC-H SF=2 (~15 M rows)
